# Blockchain & Web3 — Categories & Priority

## Categories
- PROTOCOL — L1/L2 upgrades, EIPs, consensus changes, hard forks
- DEFI — Protocol TVL, new mechanisms, liquidity, yields
- NFT — NFT markets, collections, royalties, standards
- DAO — Governance votes, treasury, proposal outcomes
- INFRA — Node infrastructure, RPC, indexers, bridges
- SECURITY — Audits, exploits, formal verification, bug bounties
- REGULATION — Blockchain-specific policy, smart contract law
- VENTURE — Funding rounds, acquisitions, ecosystem grants

## Priority Levels
- P1 MAJOR 🔴 — Major protocol upgrade (Ethereum hard fork), critical exploit >$100M, landmark ruling
- P2 SIGNIFICANT 🟠 — Protocol TVL milestone, large governance vote, major grant/fund
- P3 NOTABLE 🟡 — EIP proposed, new protocol launch, medium hack, ecosystem update
- P4 FYI ⚪ — Minor governance post, small grant, routine audit result
